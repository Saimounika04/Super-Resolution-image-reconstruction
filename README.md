# Super-Resolution-image-reconstruction using GAN
Super-Resolution Image Reconstruction Using GAN is a deep learning project that converts low-resolution images into high-resolution images using a Generator and Discriminator. It uses Content Loss, Perceptual Loss, and Adversarial Loss to improve image quality, preserve fine details, and generate realistic high-resolution images.
# Super-Resolution Image Reconstruction Using GAN

## 📌 Project Overview
Super-Resolution Image Reconstruction Using Generative Adversarial Networks (GANs) is a deep learning project that enhances low-resolution images into high-resolution images while preserving fine details and textures. The system uses a Generator to reconstruct high-quality images and a Discriminator to distinguish generated images from real images.

## 🎯 Objectives
- Improve the quality of low-resolution images.
- Generate realistic high-resolution images.
- Preserve textures and fine details.
- Reduce image blur and noise.

## 🚀 Features
- Image Super-Resolution using GAN
- Generator and Discriminator Architecture
- Content Loss
- Perceptual Loss
- Adversarial Loss
- High-quality Image Reconstruction
- Realistic Image Generation

## 🛠️ Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy
- Pandas
- Scikit-learn
- Matplotlib

## 📂 Dataset
- Flickr10K Dataset
- High Resolution Images
- Low Resolution Images generated using down-sampling

## 📊 Workflow
1. Load Dataset
2. Preprocess Images
3. Generate Low-Resolution Images
4. Train Generator Network
5. Train Discriminator Network
6. Optimize GAN using Content, Perceptual, and Adversarial Loss
7. Generate High-Resolution Images
8. Evaluate Results

## 💡 Applications
- Medical Image Enhancement
- Satellite Image Processing
- CCTV Surveillance
- Image Restoration
- Digital Photography
- Forensic Analysis

## 📈 Results
The model successfully reconstructs high-resolution images from low-resolution inputs by preserving image quality, textures, and fine details.
